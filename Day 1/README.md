## Задачи
1. Cоздать многофайловую сборку. Назначить сборке строе имя. Поместить сборку в GAC. Проанализировать полученную сборку с помощью ILDASM, JetBrains dotPeek. [Примеры в помощь](https://drive.google.com/drive/u/0/folders/0B7WmjuqYed3AWXFzc1Mtcnk3d1k)
2. Реализовать методы быстрой сортировки и сортировки слиянием для целочисленного массива (тип проекта Class Library). Протестировать работу методов с использованием тестовых фреймворков NUnit и MS Tests. Рассмотреть вариант тестирования массивов большой размерности, элементы которых сгенерированны случайным образом.
